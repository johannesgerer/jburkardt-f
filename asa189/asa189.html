ASA189
Maximum Likelihood Parameter Estimation
Beta Binomial PDF

ASA189 is a FORTRAN90 library which does a maximum likelihood estimation of the parameters of a Beta Binomial PDF, by D Smith.

ASA189 is Applied Statistics Algorithm 189. Source code for many Applied Statistics Algorithms is available through STATLIB.

This algorithm analyzes data that was generated by a beta binomial probability density function, and tries to estimate the values of the parameters that determing that PDF.

Languages:

ASA189 is available in a FORTRAN77 version and a FORTRAN90 version.

Author:

Original FORTRAN77 version by D Smith; FORTRAN90 version by John Burkardt.

Reference:

  1. D Smith,
    Algorithm AS 189: Maximum Likelihood Estimation of the Parameters of the Beta Binomial Distribution,
    Applied Statistics,
    Volume 32, Number 2, 1983, pages 196-204.

Examples and Tests:

  • asa189_prb.f90, a sample problem;
  • asa189_prb.sh, commands to compile, link and run the sample problem;
  • asa189_prb_output.txt, the output file.

List of Routines:

  • BBL calculates the beta binomial log likelihood.
  • BBME estimates MU and THETA of the beta binomial distribution.
  • BBML estimates the parameters of a beta binomial distribution
  • GDER computes derivatives of the log likelihood.
  • SET sets up the arrays RL and LM.
  • TIMESTAMP prints the current YMDHMS date as a time stamp.
