HB_TO_ST
Harwell-Boeing => Sparse Triplet Format

HB_TO_ST is a FORTRAN90 program which reads the definition of a sparse matrix from a file in Harwell-Boeing format, and writes the information into a Sparse Triplet file.

Usage:

hb_to_st hb.file st.file
where
  • hb.file is the HB file to read;
  • st.file is the ST file to create.

Examples and Tests:

5BY5 is a simple 5 by 5 matrix: 

        1.1  0.0  0.0  1.4  0.0
        0.0  2.2  0.0  0.0  0.0
        3.1  3.2  3.3  3.4  3.5
        0.0  0.0  0.0  4.4  4.5
        5.1  5.2  0.0  0.0  5.5
  • 5by5_rua.hb, the Harwell-Boeing file containing the matrix.
  • 5by5.st, is the corresponding sparse triple file;
  • 5by5_output.txt, the output file.

KERSHAW is a simple 4 by 4 matrix: 

         3  -2   0   2
        -2   3  -2   0
         0  -2   3  -2
         2   0  -2   3
  • kershaw_rua.hb, the Harwell-Boeing file containing the matrix.
  • kershaw.st, is the corresponding sparse triple file;
  • kershaw_output.txt, the output file.
