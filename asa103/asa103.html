ASA103
The Digamma or Psi Function

ASA103 is a FORTRAN90 library which evaluates the digamma or psi function, by Jose Bernardo.

ASA103 is Applied Statistics Algorithm 103. Source code for many Applied Statistics Algorithms is available through STATLIB.

Languages:

ASA103 is available in a C version and a C++ version and a FORTRAN77 version and a FORTRAN90 version and a MATLAB version

Related Data and Programs:

ASA121, a FORTRAN90 library which evaluates the trigamma function.

SPECFUN, a FORTRAN90 library which computes special functions, including the Psi or Digamma function.

TEST_VALUES, a FORTRAN90 library which contains sample values for a number of distributions.

TOMS708, a FORTRAN90 library which includes routines to evaluate the digamma function.

Author:

Original FORTRAN77 version by Jose Bernardo; FORTRAN90 version by John Burkardt.

Reference:

  1. Jose Bernardo,
    Algorithm AS 103: Psi ( Digamma ) Function,
    Applied Statistics,
    Volume 25, Number 3, 1976, pages 315-317.

List of Routines:

  • DIGAMA calculates DIGAMMA ( X ) = d ( LOG ( GAMMA ( X ) ) ) / dX
  • PSI_VALUES returns some values of the Psi or Digamma function for testing.
  • TIMESTAMP prints out the current YMDHMS date as a timestamp.
