ASA243
CDF of the noncentral T Distribution

ASA243 is a FORTRAN90 library which computes the cumulative density function (CDF) of the noncentral Student's T probability density function, by Russell Lenth.

ASA243 is Applied Statistics Algorithm 243. Source code for many Applied Statistics Algorithms is available through STATLIB.

Languages:

ASA243 is available in a C version and a C++ version and a FORTRAN77 version and a FORTRAN90 version and a MATLAB version

Related Data and Programs:

ASA005, a FORTRAN90 library which evaluates the CDF of the noncentral T distribution.

ASA076, a FORTRAN90 library which evaluates the Owen's T function, needed to compute the CDF of the noncentral T distribution.

OWENS, a FORTRAN90 library which evaluates Owen's T function.
