BETA_NC
CDF of Noncentral Beta Distribution.

BETA_NC is a FORTRAN90 library which evaluates the cumulative distribution function (CDF) of the noncentral Beta distribution.

Languages:

BETA_NC is available in a C version and a C++ version and a FORTRAN77 version and a FORTRAN90 version and a MATLAB version.
