RUN_79 - A run of CVT_BASIS

Run #79 works with 500 flow solutions in the T-Cell region. We subtract 5/3 of steady solution from 1-250, and 1/3 from 251 through 500. We DO NOT normalize the results. We use MASS MATRIX preconditioning.

  • cvt_run_79.inp, an input file that computes the energy of a set of preprocessed data for cluster size 7.
  • cvt_run_79.out, the output of the run.
