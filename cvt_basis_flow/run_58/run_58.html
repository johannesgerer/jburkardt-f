RUN_58 - A run of CVT_BASIS

Run #58 works with 500 flow solutions in the CAVITY region. We subtract 5/3 of steady solution from 1-250, and 1/3 from 251 through 500. We DO NOT normalize the PDE solutions. We use MASS MATRIX PRECONDITIONING.

  • cvt_run_58.inp, an input file that computes the energy of a set of preprocessed data for cluster size 4.
  • cvt_run_58.out, the output of the run.
  • gen_001.txt, centroid #1.
  • gen_002.txt, centroid #2.
  • gen_003.txt, centroid #3.
  • gen_004.txt, centroid #4.
