ASA063
the Incomplete Beta Function

ASA063 is a FORTRAN90 library which evaluates the incomplete Beta function, by KL Majumder and G Bhattacharjee.

ASA063 is Applied Statistics Algorithm 63. Source code for many Applied Statistics Algorithms is available through STATLIB.

Languages:

ASA063 is available in a C version and a C++ version and a FORTRAN77 version and a FORTRAN90 version and a MATLAB version.

Related Data and Programs:
