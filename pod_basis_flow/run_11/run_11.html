RUN_11
A run of POD_BASIS on INOUT_FLOW2 Data

This run of POD_BASIS takes the INOUT_FLOW2 data, applies mass matrix preconditioning, and seeks 16 POD vectors.

  • run_11.inp, the input file to POD_BASIS;
  • run_11.out, the output of the run.
