RUN_06
A run of POD_BASIS

This run of POD_BASIS takes the TCELL_FLOW data, applies mass matrix preconditioning, and seeks 16 POD vectors.

  • run_06.inp, the input file to POD_BASIS;
  • run_06.out, the output of the run.
