BLAS1_D
Basic Linear Algebra Subprograms
Level 1
Double Precision Real Arithmetic

BLAS1_D is a FORTRAN90 library which implements the Level 1 BLAS, or Basic Linear Algebra Subprograms in double precision real arithmetic.

The BLAS are a small core library of linear algebra utilities, which can be highly optimized for various architectures. Software that relies on the BLAS is thus highly portable, and will typically run very efficiently.

The Level 1 BLAS are primarily for use in vector operations. In certain cases, they may also be used to operate on the rows or columns of a two-dimensional array.

List of Routines:

  • DASUM takes the sum of the absolute values of a double precision vector.
  • DAXPY computes double precision constant times a vector plus a vector.
  • DCOPY copies a double precision vector X to a vector Y.
  • DDOT forms the dot product of two double precision vectors.
  • DNRM2 returns the euclidean norm of a double precision vector.
  • DROT applies a double precision plane rotation.
  • DROTG constructs a double precision Givens plane rotation.
  • DSCAL scales a double precision vector by a constant.
  • DSWAP interchanges two double precision vectors.
  • IDAMAX indexes the double precision array element of maximum absolute value.
  • LSAME returns TRUE if CA is the same letter as CB regardless of case.
  • XERBLA is an error handler for the LAPACK routines.
