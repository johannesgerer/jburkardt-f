FEM2D_POISSON_SPARSE_LAKE
A Problem in a Lake-shaped Region for FEM2D_POISSON_SPARSE

FEM2D_POISSON_SPARSE_LAKE is a FORTRAN90 library which defines the geometry of a lake-shaped region, as well as boundary conditions for a given Poisson problem, and is called by FEM2D_POISSON_SPARSE as part of a solution procedure.

Related Data and Programs:

FEM2D_POISSON_SPARSE, a FORTRAN90 program which solves Poisson's equation on a triangulated region, using the finite element method.

FEM2D_POISSON_SPARSE_ELL, a FORTRAN90 library which defines the geometry of an L-shaped region, as well as boundary conditions for a given Poisson problem, and is called by FEM2D_POISSON_SPARSE as part of a solution procedure.

Source Code:

  • lake.f90, the user-supplied routines to evaluate the right hand side, linear coefficient, and boundary conditions;
  • lake_nodes.txt, a text file containing a list, for each node, of its X and Y coordinates;
  • lake_nodes.png, a PNG image of the nodes.
  • lake_elements.txt, a text file containing a list, for each element, of the three nodes that compose it;
  • lake_elements.png, a PNG image of the elements.
