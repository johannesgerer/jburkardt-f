FILE_ROW_REVERSE
Reversed Copy of a File

FILE_ROW_REVERSE is a FORTRAN90 program which makes a reversed copy of a text file.

For instance, the file 

        I got up,
        I ate breakfast,
        Then I went to work.
would become 
        Then I went to work.
        I ate breakfast,
        I got up.

Usage:

file_row_reverse input_filename output_filename
